# Sistema-de-Vision-para-Robot-Colaborativo-UR10
Este repositorio tiene como finalidad proporcionar el código para operar el módulo didáctico de sistema de visión ubicado en el laboratorio de robótica de la Universidad Tecnológica de Tamaulipas Norte.

## Funciones clave

* Detección de objetos en tiempo real mediante sistema de visión
* Realización de Pick & Place de los elementos detectados mediante el UR10 

Para usar correctamente el módulo, seguir los pasos a continuación.

## 1-Preparación

Para poder hacer uso del módulo mediante la ejecución de código se necesitara la instalación de varias librerias en el entorno virtual. Se asume que el usuario cuenta con un editor de código de Python (lenguaje utilizado), pero en caso de no ser así, se sugiere el uso de VSCode por su facilidad de uso.

Una vez que se cuente con el editor de código, es necesaria la instalación de **Python 3.9.13** para una correcta compatibilidad entre librerías. Esto se puede hacer visitando la página oficial de Python para obtener la instalación en el siguiente link:

https://www.python.org/downloads/release/python-3913/
